# Code of Conduct Template

## 1. Purpose

This Code of Conduct outlines the expectations for all participants in this project/community. It is designed to create a welcoming, respectful, and inclusive environment for everyone involved.

## 2. Scope

This Code of Conduct applies to all participants, including:

* Project maintainers
* Contributors
* Community members
* Event participants
* Anyone interacting within project spaces (online or offline)

These spaces include discussion forums, issue trackers, social media, meetings, events, and any official communication channels.

## 3. Expected Behavior

Participants in this community are expected to:

* Be respectful and considerate of others
* Use welcoming and inclusive language
* Be open to feedback and constructive criticism
* Respect different viewpoints and experiences
* Show empathy towards other community members
* Focus on what is best for the community and project

## 4. Unacceptable Behavior

The following behaviors will not be tolerated:

* Harassment, bullying, or discrimination
* Offensive or derogatory comments related to gender, race, religion, disability, or identity
* Personal attacks or insulting remarks
* Threats or intimidation
* Sharing private information without permission
* Any other conduct that disrupts the community environment

## 5. Reporting Issues

If you experience or witness behavior that violates this Code of Conduct, please report it to the project maintainers or moderators.

When reporting an issue, include:

* A description of the incident
* Where and when it happened
* Any relevant evidence or screenshots (if available)

All reports will be handled confidentially.

## 6. Enforcement

Project maintainers or moderators are responsible for enforcing this Code of Conduct. Violations may result in:

* A warning
* Temporary suspension from the community
* Permanent removal from the project or community spaces

The severity of the action will depend on the nature of the violation.

## 7. Commitment

By participating in this project or community, you agree to follow this Code of Conduct and contribute to a positive and respectful environment for everyone.
